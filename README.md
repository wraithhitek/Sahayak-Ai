# Sahayak AI - The Visionary Futurist

Sahayak AI is a visionary agentic AI, designed to empower teachers and revolutionize the classroom experience. It is an AI-powered teaching assistant that helps teachers in multi-grade classrooms in India to create engaging and culturally relevant educational content.

## Features

*   **Hyper-Local Content Generation:** Generate culturally relevant educational content in multiple Indian languages.
*   **Differentiated Material Creation:** Instantly generate multiple versions of a worksheet tailored to different grade levels.
*   **Instant Knowledge Base:** Provide simple, accurate explanations for complex student questions in the local language.
*   **Visual Aid Design:** Generate simple line drawings or charts based on a teacher's description.
*   **Audio-Based Reading Assessments:** AI-powered reading fluency and pronunciation assessment using Vertex AI Speech-to-Text.
*   **AI-Powered Weekly Lesson Planners:** Structure activities and save teachers valuable time with AI-powered lesson planners.
*   **Multilingual Support:** Translate the entire page content into multiple Indian languages.

## Technology Stack

*   **Frontend:** Next.js, React, TypeScript, Tailwind CSS
*   **AI:** Google AI, Vertex AI Speech-to-Text
*   **Translation:** Google Translate API

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/sahayak-ai.git
    ```
2.  Install the dependencies:
    ```bash
    cd sahayak-ai
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Usage

To use the application, simply navigate to the home page and select one of the available features. You can also use the language dropdown to translate the page content into your preferred language.
